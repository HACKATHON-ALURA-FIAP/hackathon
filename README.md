
# Cidadão Alerta

Projeto desenvolvido para o Hack For Change FIAP + ALURA, onde tomamos a iniciativa
de criar o "Cidadão Alerta", solução projetada para aprimorar a segurança em Embu-Guaçu.


## Funcionalidades

- Alertas em tempo real
- Mapa Interativo
- Perfil de Suspeitos
- Avisos de Trânsito
- Notícias Locais
- Comunidade Segura
- Anonimato Protegido
